# Crimeboard
Web application to solve serious crimes. Add notes, pictures<sup>1</sup>, and maps<sup>1</sup> to the crime board, and connect them with wires<sup>1</sup>.

## Installing
Prerequisites:  
- Node.js  

To install:  
```npm install```  

To run:  
```npm start```

The site will be visible on port 3000.
